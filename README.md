# DOH-ConferencePosters
Public Repo for additional methods and materials as part of conference posters. Link to this repo with QR codes. 

## Repo Rules
### Conference subdir naming

Year_ConferenceAcronym_suffix 

Examples: 
* 2026_CSTE_Annual
* 2026_Databricks_Summit
* 2026_HIMMS
* 2026_NACCHO_Annual


### File naming

Year_ConferenceAcronym_PresentingAuthor_FileContents


Examples: 
* 2026_CSTE_Annual_WorshamDennis_Poster.pdf
* 2026_CSTE_Annual_WorshamDennis_DetailedMethods.pdf
* 2026_CSTE_Annual_WorshamDennis_DetailedFigures.pdf
